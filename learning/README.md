# learning/

A self-contained space for working through the **dbt Core & Cloud** curriculum from
[Analytics With Sushil](https://analyticswithsushil.com/syllabus), one topic at a time.

Everything related to this learning track lives **inside this folder** — nothing here
changes the actual `oneonone-analytics` pipeline.

## Files
- **[SYLLABUS.md](./SYLLABUS.md)** — full captured curriculum (the source of truth for what we cover).
- **[PROGRESS.md](./PROGRESS.md)** — checklist tracking which topics are done.
- **`topics/`** — one notes file per topic as we complete it.

## How we work (agreed approach)
1. Pick the next ⬜ topic from PROGRESS.md, **top to bottom**.
2. Teach it **deeply, concept-first** — the *why*, with generic examples (not tied to this repo).
3. **One topic at a time** — after teaching, save notes and wait for confirmation before moving on.
4. Save a notes file in `topics/` and mark the topic ✅ in PROGRESS.md for every topic.
